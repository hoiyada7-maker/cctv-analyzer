### 완성된 프로젝트 구조

```
cctv-analyzer/
├── analyzer.py             # 메인 파이프라인
├── motion_detector.py      # OpenCV 움직임 감지
├── gemini_client.py        # Gemini API 클라이언트 (Flash-Lite + Flash 폴백)
├── config.yaml.example     # 설정 템플릿
├── requirements.txt
└── README.md
```

### 핵심 기능 요약

| 단계 | 모듈 | 역할 |
| --- | --- | --- |
| 1. 영상 탐색 | analyzer.py | 어제 날짜 파일을 패턴 매칭으로 찾기 |
| 2. 움직임 감지 | motion_detector.py | OpenCV MOG2로 사건 구간만 추출 |
| 3. 인접 클립 병합 | analyzer.py | 파일 경계에 걸친 사건 + 연속 NVR 클립을 1번만 분석 |
| 4. 다운샘플 | motion_detector.py | ffmpeg로 640×360, 2fps (용량 1/20) |
| 5. Gemini 분석 | gemini_client.py | Flash-Lite 우선, 모호하면 Flash 폴백 |
| 6. 결과 중복 제거 | analyzer.py | 비슷한 description은 "(외 N건)"으로 통합 |
| 7. 일일 요약 | gemini_client.py | 텍스트로만 처리 (저렴) |
| 8. 알림 | analyzer.py | 텔레그램 + 이메일 + 비용 리포트 |

### 비용 절감 효과

원래 "하루 6시간 영상 × 30일" 시나리오에서:

| 방식 | 월 비용 |
| --- | --- |
| 영상 통째로 + Flash | $59 (≈ 82,000원) |
| FPS 0.5만 적용 | $12 (≈ 16,800원) |
| 이 프로젝트 (움직임 감지 + 중복 제거 + Flash-Lite) | 약 2,000~5,000원 |
| 무료 티어 활용 시 | 0원 가능 |

### 시작하는 법

bash

```
# 1. 다운로드 받은 프로젝트 폴더로
cd cctv-analyzer

# 2. 의존성 설치
sudo apt install -y ffmpeg
pip install -r requirements.txt

# 3. 설정 파일 작성
cp config.yaml.example config.yaml
# config.yaml 편집:
#   - gemini.api_key (https://aistudio.google.com 에서 발급)
#   - cameras[].path (본인 CCTV 영상 폴더)
#   - telegram.bot_token, chat_id

# 4. 테스트
python motion_detector.py /path/to/test.mp4   # 움직임 감지만
python analyzer.py 2026-05-09                  # 특정 날짜 전체 파이프라인

# 5. cron 등록 (매일 새벽 3시)
crontab -e
# 다음 줄 추가:
# 0 3 * * * cd /path/to/cctv-analyzer && /usr/bin/python3 analyzer.py >> logs/cron.log 2>&1
```
